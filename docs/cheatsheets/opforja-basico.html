opforja. Uso básico

Modelar OPM en la mesa: entrar, dibujar las dos primitivas, guardar. Lo esencial cabe en una hoja.

objeto proceso estado

Empezar

Entrar

opforja pide iniciar sesión (un solo operador). Abre en un modelo vacío, o retoma tu último trabajo persistido.

Todo se guarda en el servidor: no hay «archivo local» que perder. La dirección es opforja.sanixai.com.

Vocabulario

Las dos primitivas

OPM modela el mundo con dos cosas, y nada más:

  • Objeto — una cosa que existe (rectángulo).
  • Proceso — algo que ocurre y transforma (elipse).
  • Estado — una situación de un objeto (cápsula dentro de él).

Un proceso consume, produce o afecta objetos. Ese es todo el idioma.

Dibujar

Crear en el canvas

  • O objeto · P proceso · S estado — o desde la barra superior.
  • Click para colocar, arrastra para mover.
  • R conectar dos cosas. El enlace nace como consumo; cambia su tipo en el Inspector.

El Inspector (panel lateral) edita lo que tengas seleccionado: nombre, tipo, estados, enlaces.

A diario

Guardar

Pulsa Ctrl+S. El chip de estado te dice dónde estás:

  • Guardado — todo persistido.
  • Sin guardar · Ctrl+S (ámbar) — hay cambios pendientes. Pulsa el chip o el atajo.

Guardar es explícito: es tu acto de decir «esto es una base», no borrador en movimiento.

Tres gestos diarios

Guardar · Buscar · Comandos

Ctrl+SGuardar
Ctrl+FBuscar dentro del modelo
Ctrl+KPaleta de comandos

La paleta Ctrl+K es el atajo a todo: exportar, cambiar de vista, crear, navegar.

Sin ceremonia

Bosquejar: los apuntes

Nuevo abre un apunte: una hoja para trazar sin nombre, sin carpeta, sin exigencia de «cerrar» el modelo.

Todo nace apunte. Cuando el bosquejo madura, lo gradúas a modelo — recién ahí se pide nombre y se exige rigor.

Sin miedo

Deshacer y rehacer

Ctrl+ZDeshacer
Ctrl+Shift+ZRehacer

Funcionan en el canvas y en la edición. Explora tranquilo: casi todo es reversible.

Salir con el trabajo

Exportar para compartir

  • Ctrl+K › Exportar OPD actual como PNG.
  • Ctrl+K › Exportar todos los OPDs — descarga un .zip.
  • OPL a Markdown desde la paleta o el panel OPL.

Respaldo

Descargar y restaurar

Aunque todo vive en el servidor, puedes bajar una copia JSON del modelo desde la paleta, y restaurarla por Importar JSON en el gestor de modelos.

Útil antes de un cambio grande o para mover trabajo entre entornos.

Si algo se atasca

Rescate rápido

  • Chip «Sin guardar» — pulsa Ctrl+S antes de cerrar.
  • Vista rara o error — recarga con Ctrl+Shift+R.
  • Cambio no deseado — Ctrl+Z.
