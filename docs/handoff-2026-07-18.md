## Decisiones consolidadas

| Decisión | Motivo | Alternativa descartada |
|---|---|---|
| Una actualización se liga a la base observada. | Convierte concurrencia en conflicto explícito. | Última escritura gana o releer la base recién al hacer push. |
| Modelo, versión, especie y workspace usan confirmación atómica/CAS. | Una intención debe dejar todos sus efectos o ninguno. | Recomponer con endpoints heredados y tolerar residuos parciales. |
| Cada intención UI dispara una transición atómica o compensación explícita. | Mantiene equivalencia entre gesto humano y contrato backend. | Varias llamadas independientes ocultas tras un solo botón. |
| Una transformación de UX preserva atajos y estados observables. | Renombrar en cadena no puede secuestrar Escape ni la paleta global. | Tratar el input inline como un modal aislado del sistema. |
| Simulación muestra solo el observable que realmente existe. | “Listo”, “simulando” y “cerrado” son estados distintos. | Mezclar readiness, progreso y cierre en una misma etiqueta. |
| Una afirmación formal usa el estatus epistémico más débil que baste. | Igualdad de firma y correspondencia de enlaces son observables útiles, pero no demuestran por sí solos estructuras universales. | Nombrarlas equivalencia total, bisimulación, adjunción, fibración o pushout por analogía. |
| El cordón deriva estado desde fuentes vivas. | Evita una SSoT operativa paralela. | Daemon, base de versiones o documento actualizado a mano. |
| Un SHA distinto se clasifica por diff desplegable. | Un commit documental posterior no es deriva de producto. | Comparar solamente `HEAD !== build`. |
| `Resuelto` vive en histórico, no en activos. | La ubicación del expediente también comunica estado. | Dejar carpetas cerradas bajo `docs/bugs/BUG-*`. |

## Riesgos y límites vigentes

- El sello de la skill prueba identidad, versión, target y procedencia, pero no
  demuestra automáticamente equivalencia semántica completa con las cuatro
  SSOT. Ese control sigue siendo humano y está declarado.
- `uso-productivo.md` tiene leyes editoriales y E2E parciales, pero no un
  comparador de cada afirmación contra la interfaz. Es una frontera de
  automatización, no un defecto conocido del corpus.
- CAS evita sobrescritura silenciosa; no fusiona intención ni ofrece coedición
  simultánea tipo CRDT.
- Out-zoom, probabilidad declarada sin pesos, auth/roles v2 y los GAPs visuales
  son capacidades diferidas, no trabajo incompleto de este corte. Se abren solo
  ante demanda o gate afectado.
- `Testigo-Base` es atestación de protocolo, no firma criptográfica.
- El escaneo Socket gratuito no aplica políticas privadas de una organización.
- Chromium puede invalidar conexiones cuando Docker cambia interfaces `veth`.
  Ejecutar E2E sin despliegues/Testcontainers concurrentes y con `PW_PORT`
  propio; apagar primero cualquier Vite de desarrollo.
- El repo hermano `kora-pneuma` quedó limpio y sincronizado con
  `origin/master` en `fd284cf`; sus cambios siguen perteneciendo a su propia
  autoridad y no se publican desde esta mesa.
- La credencial del operador fue rotada durante el corte, el login fue
  comprobado y el archivo local conserva modo `600`. No copiar su contenido a
  Git, documentación, logs ni memoria.
